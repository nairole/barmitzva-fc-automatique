# Analyse automatique Barmitzva FC

Ce dépôt public contient uniquement le programme d'analyse vidéo. Il vérifie toutes les deux heures les rediffusions de Wadelytb déjà synchronisées par le site, analyse au maximum une nouvelle vidéo et publie les résultats détectés.

Le code administrateur est conservé dans le secret GitHub `ADMIN_TOKEN` et ne doit jamais être écrit dans un fichier du dépôt.

Le lancement peut aussi être déclenché manuellement depuis l'onglet **Actions**.
